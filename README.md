# lolbotpicker
Application that grab data from champion.gg to choose the best option on bot lane on game League of Legends


Compile with Visual studio 2015. Needed .NET Framework 4.5


To use, first select if you are going to play as ADC or Support. Select your partner ADC or Support and/or enemies Support and ADCs. The software will check in www.champion.gg the stats for that champion and will propose you a list of champions to play, ordered from better to worse.
